Fantasy Baseball Draft Helper

View on GitHub

A local web application built to dominate my fantasy baseball draft in the Bob Uecker Imaginary Baseball League. The app loads player projection CSVs and runs an AI-powered recommendation engine that scores every available player in real time. Recommendations are weighted across three factors: position scarcity (30%), team needs (30%), and projected statistical value (40%). The league runs rotisserie scoring with 13 teams, 23 roster spots per team, and specific position requirements (C, 1B, 2B, 3B, SS, MI, CI, 4 OF, U, 9 P). Batting categories include HR, OBP, R, RBI, and SB. Pitching categories include ERA, K, SV, WHIP, and WQS. The frontend updates in real time as picks happen, with search, filtering, color-coded recommendations, and auto-saved draft state.

OpenClaw on Cloudflare

Deployed and configured OpenClaw, an autonomous personal AI agent, on Cloudflare's edge infrastructure. The agent runs inside a Cloudflare Sandbox container, an isolated ephemeral micro-VM that prevents RCE risk. Persistent memory and conversation history are stored in Cloudflare R2. The admin interface is locked behind Cloudflare Access with Zero Trust authentication (JWT validation, identity provider integration). Set up the Anthropic API key routing through Cloudflare AI Gateway for caching, analytics, and provider flexibility. Tuned the instance type and sleep timers for cost optimization. The entire setup runs on Cloudflare Workers for about $5/month with no VPS or local hardware required.

Cloudflare Workers Sandbox Containers R2 Storage Zero Trust AI Gateway Docker

Projects

Key initiatives built during my time at Koch Industries.

Authentication & Identity Engineer

Device Trust

Deploying the PingOne Protect device trust agent across the enterprise. Phase 1 covers 1,700 users, with a full rollout targeting 150,000+. The agent runs on user workstations and ingests risk signals like OS patch level, device identity, and user authentication context. These signals feed into PingOne Protect to calculate a real-time risk score, which drives a custom-crafted authentication experience. Trusted devices get a streamlined flow. Unknown or risky devices trigger step-up MFA or are denied access entirely.

DaVinci PingOne Protect Bash SCCM JavaScript

KochID Core

Unified identity service for the enterprise. Developer self-service portal for registering OAuth/OIDC clients, managing app ownership, and configuring resource scopes. Includes interactive API Swagger pages, documentation, an IGA request portal for access governance, and a Grafana-backed insights dashboard so app owners can query authentication and audit data for their specific applications. Also features an AI assistant for auth-related questions.

Designed and built an automated failover system that redirects users from a DaVinci orchestration flow to a PingOne authentication policy during DaVinci or PingID MFA outages. Provides millions of dollars in outage redundancy by keeping authentication available when critical identity services go down.

Synthetic Transactions

Automated SSO test transactions that continuously probe the authentication stack. When a user reports an issue, synthetic transactions pinpoint exactly which service is failing, whether it is the identity provider, MFA, or a specific application integration. Leads to significantly faster SLA response times by eliminating guesswork during incidents.

SSO Monitoring Automated Testing SLA
